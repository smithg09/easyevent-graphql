## Development

We use `node` version `10.15.0`

```
nvm install 10.15.0
```

```
nvm use 10.15.0
```

The first time, you will need to run

```
npm install
```
Once installed start the server and frontend

```
npm run client
npm run server
```
## Build

Run `npm run heroku-postbuild ` to build the project and deploy to heroku
